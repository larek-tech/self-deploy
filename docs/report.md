# Отчет о тестировании скрипта (метрики)

## Исходные данные

Для тестирования был выбран расширенный [датасет](https://drive.google.com/drive/folders/18iqoXmqrCEKZJEu4x23Hw1O6FX7l5GzI?usp=sharing) репозиториев на основных языка: Go, Pytho, Java/Kotlin, JS/TS.\n
Были выставлены ограничения проверки: максимальный размер репозитория - 2МБ, количество репозиториев/язык - 100.\n

## Тестирование

Для тестирования подготовлены два скрипты - [clone](../clone.py) и [metrics](../metrics.py).\n

- [clone](../clone.py) клонирует репозитории из датасета (необходимо расположить .csv файлы в корне проекта в папке data/data)
- [metrics](../metrics.py) считает метрики *precision*, *recall* и *f1* на основе отчета скрипта и реального расположения файлов в репозитории.
