# RepoSchema — автосгенерированная документация

Описание полей модели RepoSchema:

| Поле | Тип | Обязательное | Описание |
|---|---|---|---|
| `is_monorepo` | `boolean` | Да | Флаг, указывающий, является ли репозиторий монорепой. |
| `services` | `array` | Нет | Список сервисов в репозитории |
| `deployment` | `` | Да | Информация о деплое сервиса |

## Пример JSON (скелет)
```json
{
  "is_monorepo": null,
  "services": null,
  "deployment": null
}
```
